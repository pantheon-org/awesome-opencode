# Rejected Submissions

This document tracks all submissions that were rejected during the triage process. This serves as:

- A duplicate detection mechanism
- Transparency for contributors
- An audit trail for decisions

## Rejection Reasons

| Reason              | Description                                                           |
| ------------------- | --------------------------------------------------------------------- |
| `not-relevant`      | Tool does not align with AI coding assistant or related tooling scope |
| `duplicate`         | Repository already submitted (see existing entry)                     |
| `archived`          | Repository is archived and no longer maintained                       |
| `spam`              | Submission identified as spam or promotional abuse                    |
| `invalid-url`       | URL is not accessible or does not point to a valid GitHub repository  |
| `insufficient-info` | Repository lacks adequate documentation or information                |
| `commercial-only`   | Tool is closed-source commercial-only without open-source component   |
